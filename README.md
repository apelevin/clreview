# Обзор судебной практики

MVP сервиса для автоматической обработки документов судебной практики и создания обзоров.

## Возможности

- Загрузка множественных DOCX файлов
- Автоматическое преобразование документов в правовые позиции
- Создание единого обзора судебной практики на основе всех документов
- 5-шаговый pipeline обработки с использованием OpenRouter API

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env.local` на основе `.env.example`:
```bash
cp .env.example .env.local
```

3. Добавьте ваш OpenRouter API ключ в `.env.local`:
```
OPENROUTER_API_KEY=your_openrouter_api_key_here
```

Опционально, вы можете добавить дополнительные заголовки для отслеживания:
```
OPENROUTER_HTTP_REFERER=https://github.com/apelevin/review
```

## Запуск

Разработка:
```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

Сборка для продакшена:
```bash
npm run build
npm start
```

## Использование

1. Откройте приложение в браузере
2. Загрузите один или несколько DOCX файлов с судебными документами
3. Дождитесь завершения обработки (может занять несколько минут)
4. Просмотрите созданный обзор и при необходимости скачайте его

## Структура проекта

- `app/` - Next.js приложение (страницы и API routes)
- `lib/` - утилиты (конвертация DOCX, LLM клиент, pipeline)
- `prompts/` - системные промпты для обработки документов
- `uploads/` - временное хранилище загруженных файлов
- `processed/` - обработанные документы и результаты

## Технологии

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- OpenRouter API
- Mammoth (конвертация DOCX)
